## Overview

The `HomeScreen` widget is the main interface of the PhotoVoice app, displaying a real-time feed of posts and providing access to camera functionality with ethical consent prompts.

**Location:** `lib/features/posts/presentation/screens/home_screen.dart`

## Key Features

### 1. Real-time Post Feed

Displays posts using a `StreamBuilder` connected to `PostProvider.postsStream` for automatic updates.

From `home_screen.dart:180-257`:

```dart
final postStream = context.read<PostProvider>().postsStream;

return StreamBuilder<List<Post>>(
  stream: postStream,
  builder: (context, snapshot) {
    if (snapshot.connectionState == ConnectionState.waiting) {
      return const Center(child: CircularProgressIndicator());
    }

    if (snapshot.hasError) {
      return Center(
        child: Text('Sin conexión.\nPor favor, revisa tu red.'),
      );
    }

    final posts = snapshot.data;
    
    if (posts == null || posts.isEmpty) {
      return Center(
        child: Text('No hay voces registradas aún.\nSé el primero en compartir tu realidad.'),
      );
    }

    return RefreshIndicator(
      onRefresh: _onRefresh,
      child: ListView.builder(
        itemCount: posts.length,
        itemBuilder: (context, index) {
          final post = posts[index];
          return PostCard(post: post);
        },
      ),
    );
  },
)
```

### 4. Data Export (CSV)

Exports all posts to CSV format for qualitative research analysis.

From `home_screen.dart:95-146`:

```dart
Future<void> _exportData(BuildContext context) async {
  try {
    ScaffoldMessenger.of(context).showSnackBar(
      const SnackBar(content: Text('Generando archivo CSV...')),
    );

    final provider = Provider.of<PostProvider>(context, listen: false);
    final List<Post> posts = await provider.fetchAllPosts();

    // Create CSV headers
    List<List<dynamic>> csvData = [
      ['Fecha', 'URL_Imagen', 'Narrativa_Reflexion', 'ID_Participante'],
    ];

    // Transform posts to CSV rows
    for (var post in posts) {
      csvData.add([
        post.createdAt.toIso8601String(),
        post.imageUrl,
        post.comment,
        post.id,
      ]);
    }

    // Convert to CSV string
    String csv = const ListToCsvConverter().convert(csvData);

    // Write to temporary file
    final directory = await getTemporaryDirectory();
    final path = '${directory.path}/photovoice_export_${DateTime.now().millisecondsSinceEpoch}.csv';
    final File file = File(path);
    await file.writeAsString(csv);

    // Share file
    await Share.shareXFiles(
      [XFile(path)],
      text: 'Exportación de datos de codificación cualitativa - PhotoVoice',
      subject: 'PhotoVoice_CSV_Export',
    );
  } catch (e) {
    if (!context.mounted) return;
    ScaffoldMessenger.of(context).showSnackBar(
      SnackBar(
        content: Text('Error al exportar los datos: $e'),
        backgroundColor: Colors.redAccent,
      ),
    );
  }
}
```

### _openCamera()

**Output format:**

- Filename pattern: `photovoice_export_{timestamp}.csv`

## State Management

The screen is stateless and relies on:
- `PostProvider` for post data and state
- `StreamBuilder` for reactive UI updates
- Provider package for dependency injection
